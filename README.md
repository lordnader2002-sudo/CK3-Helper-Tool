# CK3 AGOT Helper Tool

A browser-based strategy advisor for **Crusader Kings 3** with the **A Game of Thrones** mod. Open `index.html` in any browser — no installation required.

## Features

### Houses of Westeros
- All 8 major houses with stats, strengths, weaknesses, and strategic tips
- Key vassals, starting rulers, difficulty ratings, Economy/Military/Diplomacy ratings
- Color-coded house banners and detailed drill-down views

### Character Optimizer
- Input your ruler's 5 core stats + Prowess, Age, and Prestige level
- Select your playstyle (Conqueror / Diplomat / Schemer / Economy / Faith)
- Get personalized lifestyle focus recommendations, council advice, and combat assessment

### War Planner
- Army composition calculator (levies + men-at-arms vs enemy)
- Terrain type modifiers (Plains, Hills, Forest, Desert, Mountains, River Crossing)
- Commander quality impact and special factors (dragons, holy wars, winter, mercenaries)
- Full Men-at-Arms counter guide for all unit types including AGOT-specific (Dothraki, Unsullied)

### Succession Planner
- All 6 succession laws compared with pros/cons
- Realm integrity, predictability, and risk ratings
- Visual comparison table + succession tips

### Intrigue & Schemes
- Scheme success calculator for Murder, Kidnapping, Claim Throne, Seduction
- Agent, Spymaster, hooks, and target condition modifiers
- Full scheme reference guide with per-scheme tips

### Buildings
- All key buildings with type, slot, effect, and priority rating
- Filter by holding type and priority level
- Region-specific building advice for all 12 regions of Westeros + Essos

### Traits Reference
- Personality traits (positive/negative/mixed) with stat effects
- Lifestyle perks with tier info
- AGOT-unique traits (Dragonlord, Warg, Greenseer, Valyrian Steel, etc.) with rarity ratings

### Pro Tips
- 8 strategy categories: Early Game, Vassal Management, Economy, Military, Intrigue, AGOT-Specific, Diplomacy, Dynasty Power

## Usage

1. Clone or download this repository
2. Open `index.html` in your browser
3. No server required — fully static HTML/CSS/JS
